# Boykot Mobile App

A Flutter mobile application that helps users identify brands to boycott based on data scraped from public sources.

## Features

- Browse boycott information for various brands and companies
- View detailed reasons for boycotts
- Access up-to-date data from public sources
- Multi-platform support (iOS, Android, Web, Windows, Linux, macOS)

## Getting Started

### Prerequisites

- Flutter SDK (version 3.9.2 or higher)
- Dart SDK

### Installation

1. Install dependencies:
   ```bash
   flutter pub get
   ```

2. Run the app:
   ```bash
   flutter run
   ```

## Key Dependencies

- `http`: For making network requests
- `url_launcher`: For opening external links
- `google_fonts`: For typography
- `intl`: For internationalization

## Data

The app consumes boycott data from `assets/boycott_data.json`, which is generated by the backend scraper located in the `../backend` directory.

## Project Structure

- `lib/`: Main application code
  - `main.dart`: Application entry point
  - `app.dart`: App configuration
  - `screens/`: UI screens
  - `widgets/`: Reusable widgets
  - `services/`: Business logic and services
  - `utils/`: Utility functions
- `assets/`: Static assets including data and images
